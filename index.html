Writing CSS is tough stuff!

Tailwind makes writing css easier and lets u customize our designs faster than any other css framework. If you want a css framework that has pre-built components, then Tailwind may not be for you. If you want a css framework that lets you design quickly and customize your stie, then Tailwind is gonna be a lot of fun.

After using Bulma to build Scotch.io, my css files became littered with my own !important classes to override the framework' styles. Tailwind doesn't have the !important problem since all our elements are designed on the fly.

Important Note: Tailwind let us design quickly, but this doesn't mean that you don't need css knowledge however. To use Tailwind, you need to have a good understanding off css properties. in this course, we will learn Tailwind and we'll also learn css concept to create good looking designs. Tailwind is the ultimate css shorthand, but we still need to learn the css, which we'll do in this scourse.

"This isn't just a Tailwind course. This is also a "learn how to design with css" course.

The hardest thing about learning Tailwind is memorizsing the classes and also knowing which clases go well together. We'll be writing a lot of Tailwind in thi course so that the repetition will help us commit Tailwind clases to memory.
